# Equipo 61 – Avance 1: Análisis Exploratorio de Datos

## Descripción del proyecto

Este repositorio contiene el desarrollo del **Avance 1: Análisis Exploratorio de Datos (EDA)** correspondiente al proyecto integrador enfocado en la inferencia de propiedades físicas de galaxias a partir de datos astronómicos.

El análisis fue realizado sobre un conjunto de datos astronómico que incluye variables físicas, espectrales y morfológicas de galaxias, con el objetivo de identificar patrones, relaciones estadísticas y posibles problemas de calidad de datos antes del desarrollo de modelos de aprendizaje automático.

## Actividades realizadas

Durante el análisis exploratorio se realizaron las siguientes actividades:

- Revisión de estructura y tipos de datos
- Identificación de valores faltantes y placeholders
- Estadísticas descriptivas
- Análisis univariante mediante histogramas y boxplots
- Evaluación de asimetría y valores atípicos
- Análisis de correlación y relaciones multivariantes
- Estrategias de preprocesamiento:
  - Imputación de valores faltantes
  - Transformación Yeo-Johnson
  - Escalamiento de variables
- Interpretación de resultados dentro del contexto astronómico

## Objetivo del análisis

El propósito de este análisis exploratorio es generar una base sólida para futuras etapas de modelado predictivo orientadas a la inferencia de propiedades físicas de galaxias mediante técnicas de aprendizaje automático.
